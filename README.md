# netology-hw-list

Решение домашнего задания к занятию «Коллекции List».

## Реализация

Написал класс, `Task`, который:

1) Инкапсулирует в себе приватное поле - ссылку на список String, который и будет содержать задачи, согласно условию. Инициализируется конструктором.

2) Содержит методы, согласно условию задачи (укажу с сигнатурой и типами возвращаемых значений):
- `public void add(String task)` - добавляет дело в соответствующий список экземпляра класса Task;
- `public void showAllTasks()` - выводит на экран список дел с нумерацией;
- `public boolean removeByID(int num)` - удаление дела по его номеру;
- `public boolean removeByTask(String task)` - удаление дела по его точному тексту;

Методы, реализующие удаление, возвращают `boolean` - решил сделать таким образом, чтобы использовать в логике диалога с пользователем.
3) Дополнительно, в рамках задания 2 (необязательное), в классе `Task` описал метод, который удаляет дело из списка по ключевому слову:
- `public boolean removeByPartValue(String part)`
